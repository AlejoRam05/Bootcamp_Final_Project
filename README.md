# Bootcamp_Final_Project

# Descripcion del Proyecto

Este proyecto es un análisis de datos realizado en Jupyter Notebook, donde se documenta paso a paso el proceso de limpieza, exploración, modelado y visualización de un conjunto de datos. Se utilizan herramientas de Python para extraer insights relevantes, que ayudaran a enterder el concepto del negocio.

# Dataset Utilizado

- Fuente: Proporcionado por el guia del curso.
- Formato: CSV
- Descripcion: Registro de ventas del año 2019. contiene los productos vendidos, cantidad de ventas,  valores de venta y registro de las cuidades donde se entregaron.

# Metodologia y Analisis Realizado

1. Carga y Exploración de Datos  
- Lectura del dataset con Pandas  
- Análisis exploratorio de datos (EDA)

2. Enriquecimiento de Datos  
- Transformaciones necesarias

3. Visualización de Datos

- Uso de Matplotlib y Seaborn para gráficos

# Requisitos y Configuración
Asegúrate de tener Python 3.8+ y Jupyter Notebook instalado. Luego, instala las librerías necesarias ejecutando:  
> pip install -r requirements.txt  

luego abre el archivo .ipynb y sigue los pasos documentados en las celdas.

# Resultados y Conclusiones

- Los principales hallazgos fueron: 
1. El negocio tuvo un crecimiento continuo a lo largo del año. 
2. Vende muchos productos muy baratos. 
3. Vende pocos productos caros
4. Cierra el año con grandes ganancias
- Posibles mejoras:
1. Incentibar la compra ofreciendo descuentos
2. Ofrecer nuevos productos
3. Crear una base de datos para sus clientes recurrentes o nuevos.
